# 📊 Sales Analysis & Change Detection

## 🔍 Overview

Analyzed 50 weeks of transaction data to identify trends, detect a sudden shift in sales, and validate the change using statistical testing.

---

## ⚙️ Tools Used

* Power BI (Dashboard & analysis)
* Excel (T-test for statistical validation)

## 📈 Key Insights

* A significant increase in daily sales observed around **May 2013**
* Sales increased from ~1000 to ~1400+ transactions per day
* **P-value = 6.58E-119** → change is statistically significant
* Male customers contributed more to the increase
* Afternoon accounts for the highest share of sales (~38%)
